    Streamlit Para Ciência de Dados¶

    "A forma mais rápida de construir e compartilhar aplicativos de dados"

    Streamlit é um framework para desenvolvimentos de aplicações web em python de forma rápida e interativa.

    Possui convenientes métodos built-in que permitem input de usuários, e a criação e implementação de gráficos usando as bibliotecas mais populares de Python.

    Instalação¶

    Podemos simplesmente instalar com pip:

    pip install streamlit

    Podemos rodar uma demonstração escrevendo no terminal:

    streamlit hello

    O que irá abrir uma janela no navegador rodando uma aplicação Streamlit em servidor local.

    Introdução¶

    Streamlit tem funções únicas para cada tipo de conteúdo como textos, gráficos, figuras e outros tipos de mídia, usamos estes conteúdos como blocos para todos os aplicativos.

    A função st.write() leva uma string (dentre outros tipos de objetos do python) e escreve ela diretamente no aplicativo, na ordem em que ela é chamada.

    Hello World¶

    A aplicação mais básica que pode ser construída é a que simplesmente printa "Hello World" na tela.

    import streamlit as st
st.write('Hello World')

    Para rodar uma aplicação no Terminal:

    streamlit run clt_demo.py

    st.pyplot()¶

    A função st.pyploy() no permite usar a biblioteca matplotlib no nosso aplicativo. Para mostrar o gráfico na página usamos a figura criado como argumenta para a função.

    import streamlit as st
import numpy as np
import matplotlib.pyplot as plt

points = np.linspace(-5,5,100)
sigmoids = [ (1/(1+(np.exp(-x)))) for x in points]

fig, ax = plt.subplots()
ax = plt.plot(points, sigmoids)
st.pyplot(fig)

    User Input¶

    Streamlit tem várias funções para entrada de usuários, como st.text_input() para texto de forma livre, st.radio() e st.number_input() para números.

    Exemplo: plot_alura.py

    import streamlit as st
import matplotlib.pyplot as plt

st.write("Qual a sua altura comparada à minha?")
your_height = st.number_input(label= "Sua altura em centímetros")

fig, ax = plt.subplots()
ax = plt.bar(x = ["me", "you"], height=[177, your_height])
st.pyplot(fig)

    Adicionando texto em Streamlit¶

    Além de st.write, outras funções ajudam a criar a estrutura de texto do aplicativo:

    • st.title()
    • st.header()
    • st.subheader()
    • st.markdown()

    Exploração e Manipulação de Dados¶

    Usando a função select_box() pode receber um input do usuário para selecionar um item dentre uma lista de opções, o primeiro argumento é o título do campo que irá aparecer no aplicativo e o segundo argumento é a lista de opções.

    Com o próximo aplicativo, podemos criar scatterplots personalizados dos atributos dos Pokemons

    import streamlit as st
import matplotlib.pyplot as plt
import pandas as pd
import seaborn as sns

path = "https://gist.githubusercontent.com/armgilles/194bcff35001e7eb53a2a8b441e8b2c6/raw/92200bc0a673d5ce2110aaad4544ed6c4010f687/pokemon.csv"

poke_df = pd.read_csv(path)

st.title("Explorando Pokemons")
st.subheader("Um app para explorar status dos Pokemons com scatterplots")

poke_type_1 = list(poke_df["Type 1"].unique())
atributos = ['HP', 'Attack', 'Defense', 'Sp. Atk', 'Sp. Def', 'Speed']

selected_type = st.selectbox("Selecione um tipo de pokemon", poke_type_1)
selected_x = st.selectbox("Selecione um atributo para o eixo x", atributos)
selected_y = st.selectbox("Selecione um atributo para o eixo y", atributos)

data = poke_df[poke_df["Type 1"] == selected_type]

fig, ax = plt.subplots()
ax = sns.scatterplot(data = data, x = selected_x, y = selected_y, hue="Legendary")
plt.xlabel(selected_x)
plt.ylabel(selected_y)
st.pyplot(fig)

    Basicamente, usamos pandas para manipulação básica de dados com st.selectbox() nos fornecendo a entrada desejada.

    Upload de Arquivos¶

    A função st.file_uploader() permite o upload de arquivos para nossa aplicação. O tamanho máximo é de 200mb.

    Podemos usar para ler um conjunto de dados, por exemplo.

    arquivo = st.file_uploader('Selecione seu arquivo local')
dataframe = pd.read_csv(arquivo)

    Visualização de Dados com Streamlit¶

    Além da funções nativas de Streamlit para visualização de dados, podem ser usadas as seguintes biblioteca de Python:

    • Matplotlib

    • Plotly
    • Bokeh
    • Altair
    • PyDeck

    Funções Nativas¶

    Existem 3 funções nativas no Streamlit para criação de gráficos:

    • st.line_chart()
    • st.bar_chart()
    • st.area_chart()

    O código abaixo pode ser adicionado ao último app criado para testar as funções:

    st.write("Testando as funções nativas")

contagem_tipos = poke_df['Type 1'].value_counts()

st.area_chart(contagem_tipos)
st.line_chart(contagem_tipos)
st.bar_chart(contagem_tipos)

    As funções nativas criam gráficos interativos que possuem um visual e formatação bacana (usando Altair como backend). Estas funções são bastante úteis para problemas simples, porém bem menos flexíveis do que as bibliotecas próprias para este fim.

    Outra função, st.map() utiliza PyDeck e plota mapas em DataFrame com colunas do tipo longitude e latitude. Cada linhas será um ponto no mapa.

    Plotly¶

    A função st.plotly_chart() permite que sejam chamados gráficos de plotly.

    Usando plotly para explorar a distribuição dos atributos:

    selected_att = st.selectbox("Selecione um atributo", atributos)

fig = px.histogram(data[selected_att])
st.plotly_chart(fig)

    Machine Learning com Streamlit¶

    O código abaixo cria um modelo básico para classificar pokemons como Lendário ou não, e então salva o modelo com pickle.

    import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import OneHotEncoder
from sklearn.preprocessing import StandardScaler
from sklearn.ensemble import RandomForestClassifier
from sklearn.pipeline import Pipeline
from sklearn.metrics import accuracy_score, roc_auc_score, classification_report

poke_df = pd.read_csv("https://gist.githubusercontent.com/armgilles/194bcff35001e7eb53a2a8b441e8b2c6/raw/92200bc0a673d5ce2110aaad4544ed6c4010f687/pokemon.csv")

# Processo básico de modelagem
X = poke_df[['Total', 'HP', 'Attack', 'Defense', 'Sp. Atk', 'Sp. Def', 'Speed']]
y = poke_df['Legendary']

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.33, random_state=42)

pipe = Pipeline([
    ('scaler', StandardScaler()), 
    ('model', RandomForestClassifier(random_state = 42))
    ])
pipe.fit(X_train, y_train)

# Salvando o modelo
import pickle
filename = 'poke_model.sav'
pickle.dump(pipe, open(filename, 'wb'))

    Então podemos carregar o modelo na aplicação Streamlit e testar se está funcionando.

    filename = "poke_model.sav"
loaded_pipe = pickle.load(open(filename, 'rb'))

X = poke_df[['Total', 'HP', 'Attack', 'Defense', 'Sp. Atk', 'Sp. Def', 'Speed']]
y = poke_df['Legendary']

X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.33, random_state=42)
y_pred = loaded_pipe.predict(X_test)

acc = accuracy_score(y_test, y_pred)
auc = roc_auc_score(y_test, y_pred)

st.subheader("Carregando um modelo de ML pré treinado")
st.write(f"Acurácia: {acc}")
st.write(f"AUC score: {auc}")
st.text(classification_report(y_test, y_pred))

    Deploy com Streamlit¶

    Podemos usar Streamlit Cloud para implementar, gerenciar e compartilhar em apenas 3 passos:

    1. Colocar o app em um repositório público do GitHub (incluindo requirements.txt)
    2. Se cadastrar em share.streamlit.io
    3. Clicar em "Deploy an App" e então compartilhar a URL do GitHub.
