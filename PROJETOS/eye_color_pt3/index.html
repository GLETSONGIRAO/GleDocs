    Parte 3: Criação do Modelo¶

    Introdução¶

    Finalmente chegamos à parte mais simples do projeto: criar um modelo preditivo.

    Para esta tarefa foram usadas duas abordagens, usar todos os dados e desconsiderar dados em que a snp rs12913832 esteja faltando, visto que as predições sem esse atributo podem não ser confiáveis devido ao seu alto valor preditivo.

    Modelos utilizados:

    1. Regressão Logística
    2. Random Forest
    3. XGBoosting

    Procedimentos Utilizados:

    1. One Hot Encoding nas features categóricas (todas).

    2. Separação em treino e teste (20%).

    3. RepeatedStratifiedKFold com 5 splits e 3 repetições.

    4. Otimização de Hiperparâmetros com Grid Search ou Random Search

      • Grid Search com Validação Cruzada para Regressão Logística.

      • Random Search com Validação Cruzada para Random Forest e XGBoosting.

    Critério para encontrar o melhor modelo: Modelo que apresentar a melhor métrica acurácia.

    Considerações Finais¶

    Apesar da complexidade dos modelos escolhidos e do grande espaço de busca percorrido na tarefa de otimização de Hiperparâmetros, o modelo com melhor resultado também foi o modelo mais simples, Regressão Logística.

    Resumo dos resultados

    Modelo Acurácia
    Regressão Logística 79.82%
    Random Forest 78.92%
    XGBoosting 78.03%

    Apesar da baixa capacidade do modelo de classificar cores intermediárias, existe uma grande chance de que o modelo faça a predição correta de uma nova amostra, demonstrando a viabilidade da execução da tarefa utilizando Aprendizado de Máquina. Atingindo assim o objetivo da criação deste projeto.

    Alguns fatores que afetam a performance do modelo devem ser considerados:

    • Modelos preditivos para cor do olho funcionam melhor se forem treinados na população específica em que vai ser utilizado, os dados utilizados aqui contém amostras de todo o mundo.
    • O preenchimento do campo de cor do olho na fonte de coleta dos dados é subjetivo.
    • O mapeamento da cor do olho preenchida para um grupo de de cor do olho feito por mim também foi subjetivo.
