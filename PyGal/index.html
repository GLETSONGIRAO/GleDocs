    PyGal

    Início rápido¶

    PyGal é uma biblioteca python para criação de gráficos dinâmicos em SVG.

    Instalação¶

    Simplesmente usar pip.

    $ pip install pygal

    Usar em Jupyter Notebooks¶

    from IPython.display import HTML

    Exemplos básicos¶

    Coisas que dão Medo (Pesquisa elaborada pelo seríssimo Centro de Pesquisas CTPK).¶

    Em um Jupyter Notebook:

    from IPython.display import HTML

bar_chart = pygal.Bar(truncate_legend=25)
bar_chart.title = 'Coisas que mais dão medo (CTPK - 2022)'
bar_chart.add("Barata Voadora", [30])
bar_chart.add("Dois Caras Numa Moto", [45])
bar_chart.add("Entrar no próximo Leiófe", [24])
bar_chart.add("Resultado da eleição 2022", [83])
bar_chart.render_to_file('coisas_que_dao_medo.svg')

HTML(''' <object data="coisas_que_dao_medo.svg" width=100% height=100%> </object> ''')

    Com lxml instalado, o comando render_in_browser faz o Gráfico aparecer no navegador padrão.

    bar_chart.render_in_browser()

    Os componentes de um gráfico de pygal são bastante intuitivos, neste Gráfico podemos alguns dos principais elementos:

    • A documentação de pygal adota este padrão para a nomeação dos objetos: tipodegráfico_chart.
    • Os objetos de gráfico estão dentro do pacote pygal e podem ser inicializados usando a sintaxe: pygal.Tipo(parâmetros), um exemplo de parâmetro é o truncate_legend, que usei neste gráfico para aumentar o número de caracteres na legenda antes que sejam truncados.
    • chart.add usado para adicionar os elementos ao gráfico.
    • chart.render() método para renderizar o SVG. Em um jupyter notebook precisamos utilizar display e SVG.

    A função chart.render_to_file('chart.svg') pode ser usada para criar um arquivo SVG que pode facilmente ser colocado na WEB como este gráfico acima.

    A função HTML de IPython.display serve para renderizar HTML no IPython.

    Para adicionar o gráfico com interatividade em uma página html use a tag object:

    <object data="assets/browser_evolution.svg" width=80% height=50%> </object>

    Forma Alternativa de escrever o código do gráfico acima:

    # escrever uma vez
import pygal as pg
from IPython.display import HTML
import pygal as pg

def pgr(chart_object, nome_arquivo):
    """
    Função para renderizar arquivos SVG criados por pygal em um jupyter notebook, 
    ela cria uma pasta charts na pasta de trabalho salva o SVG nela e sem seguida,
    renderiza o SVG em um HTML usando a tag <object>, que é a tag apropriada para SVG dinâmico. 

    args: 
        - chart_object (um objeto de gráfico em pygal que queremos renderizar)
        - nome_arquivo (o nome)
    return:
        - SVG dinâmico renderizado em HTML.

    """
    import os
    from IPython.display import HTML
    os.makedirs('charts', exist_ok=True) 

    chart_object.render_to_file(f"charts/{nome_arquivo}.svg")

    return HTML(''' <object data="charts/{}.svg" width=95% height=80%> </object> '''.format(nome_arquivo))

# Código para o gráfico interativo
c = pg.Bar(truncate_legend=25)
c.title = 'Coisas que mais dão medo (CTPK - 2022)'
c.add("Barata Voadora", [30])
c.add("Dois Caras Numa Moto", [45])
c.add("Entrar no próximo Leiófe", [24])
c.add("Resultado da eleição 2022", [83])
pgr(c, 'medo')
