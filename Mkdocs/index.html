    Mkdocs

    Introdução¶

    Referência rápida para Material for MkDocs, um tema de MkDocs, um gerador de site estáticos direcionado a documentação técnica de projetos.

    No meu projeto, biblioteca mkdocs-material foi instalado no conda enviroment docs, que é um clone do base miniconda com alguns pacotes básicos preciamente instalados, como python e pip.

    conda create --name docs --clone base

    conda activate docs

    Instalação¶

    pip install mkdocs-material

    ### Isso cria a seguinte estrutura:
.
├─ docs/
│  └─ index.md
└─ mkdocs.yml

    Criando um site¶

    No diretório em que está o projeto:

    mkdocs new .

    Configuração¶

    Instalar o tema 

    theme:
  name: material

    Visualizar o site ao vivo¶

    mkdocs serve

# localhost:8000

    Construindo o site¶

    mkdocs build

    Publicando o site¶

    1. Criar um repositório no github.
    2. Fazer o push do diretório do projeto.
    3. Utilizar o comando: 
    mkdocs gh-deploy --force

    Customização¶

    CSS adicional¶

    Criar um novo arquivo css em stylesheets.

    .
├─ docs/
│  └─ stylesheets/
│     └─ extra.css
└─ mkdocs.yml

    Adicionar os parâmetros em mkdocs.yml:

    extra_css:
  - stylesheets/extra.css

    JavaScript Adicional¶

    .
├─ docs/
│  └─ javascripts/
│     └─ extra.js
└─ mkdocs.yml

    Adicionar os parâmetros em mkdocs.yml:

    extra_javascript:
  - javascripts/extra.js

    Mudar Cores¶

    • scheme:
    • default = Light mode
    • slate = Dark mode 
    theme:
  palette:
    scheme: default
    • Cor primária:
    • red, pink, purple... 
    theme:
  palette:
    primary: indigo

    Mudar fonte¶

    Pode ser colocada qualquer fonte do Google Fonts

    theme:
  font:
    text: Roboto
    code: Roboto Mono

    Mudar logo¶

    1. Criar pasta assets dentro no diretório com os markdowns.
    2. Colocar ícone desejado dentro da pasta asset.
    3. Incluir os parâmetros no mkdocs.yml:
    theme:
  logo: assets/logo.png

    Mudar a url para que o logo aponta:

    extra:
  homepage: https://example.com

    Estutura de Navegação¶

    Expandir todos os subtópicos automaticamente:

    theme:
  features:
    - navigation.expand

    Habilitar Busca¶

    plugins:
  - search:
      lang: pt

    Footer¶

    extra:
  social:
    - icon: fontawesome/brands/twitter 
      link: https://twitter.com/squidfunk

    Sintaxe:

    Começa com !!! seguido do quantificador de tipo, o texto vai identado por 4 espaços na próxima linha.

    !!! note
    lorem ipsum dolor

    Mudando o título:

    !!! note "Título desejado"
    lorem ipsum dolor

    Admonition colapsável:

    ??? note "Título desejado"
    lorem ipsum dolor

    Content tabs¶

    Podemos agrupar texto, código e etc...

    markdown_extensions:
  - pymdownx.superfences
  - pymdownx.tabbed:
      alternate_style: true

    int main(void) {
      printf("Hello world!\n");
      return 0;
    }

int main(void) {
  std::cout << "Hello world!" << std::endl;
  return 0;
}

    Ícones e Emojis¶

    Podem ser usados mais de 8000 ícones e milhares de emojis já integrados.

    Ativação:

    markdown_extensions:
  - attr_list
  - pymdownx.emoji:
      emoji_index: !!python/name:materialx.emoji.twemoji
      emoji_generator: !!python/name:materialx.emoji.to_svg

    Imagens¶

    configuração:

    markdown_extensions:
  - attr_list
  - md_in_html

    Alinhamento:

    # esquerda
![Image title](https://dummyimage.com/600x400/eee/aaa){ align=left }
# direita
![Image title](https://dummyimage.com/600x400/eee/aaa){ align=right }

    Subtítulo:

    <figure markdown>
  ![Image title](https://dummyimage.com/600x400/){ width="300" }
  <figcaption>Image caption</figcaption>
</figure>

    Matemática¶

    Precisamos ativar MathJax:

    Em docs/javascripts/mathjax.js

    window.MathJax = {
  tex: {
    inlineMath: [["\\(", "\\)"]],
    displayMath: [["\\[", "\\]"]],
    processEscapes: true,
    processEnvironments: true
  },
  options: {
    ignoreHtmlClass: ".*|",
    processHtmlClass: "arithmatex"
  }
};

document$.subscribe(() => { 
  MathJax.typesetPromise()
})

    Em mkdocs.yml

    markdown_extensions:
  - pymdownx.arithmatex:
      generic: true

extra_javascript:
  - javascripts/mathjax.js
  - https://polyfill.io/v3/polyfill.min.js?features=es6
  - https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-mml-chtml.js
